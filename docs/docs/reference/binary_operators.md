# Binary Operators

**Grammar**
```
(operator left-operand right-operand)
```

* **Arithmetic** (`*`, `/`, `%`, `+`, `-`)
* **Bit shifting** (`<<`, `>>`, `>>>`)
* **Comparison** (`<`, `>`, `<=`, `=>`)
* **Equality** (`==`, `!=`, `===`, `!==`)

## Instanceof

Instanceof is used to check whether an instance is of some class type.

**Grammar**
```
(instanceof instance class-type)
```

**Example**
```lisp
(instanceof x Object)
```

```js
(x instanceof Object)
```
